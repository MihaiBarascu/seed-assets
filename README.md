# Seed Assets

Public repository for seed images used in Payload CMS website templates.

## Structure

```
seed-assets/
├── transilvania-fitness/    # Gym/Fitness website template
│   ├── trainers/           # Trainer profile images
│   ├── classes/            # Fitness class images
│   ├── hero/               # Hero section backgrounds
│   ├── blog/               # Blog post images
│   └── subscriptions/      # Subscription/pricing images
└── [other-projects]/       # Future projects
```

## Usage

Raw URLs format:
```
https://raw.githubusercontent.com/MihaiBarascu/seed-assets/main/[project]/[category]/[image].jpg
```

Example:
```
https://raw.githubusercontent.com/MihaiBarascu/seed-assets/main/transilvania-fitness/trainers/trainer-male-1.jpg
```
